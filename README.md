## Geomask

[ [傳送門↗](http://maps.vasile.ch/geomask/) ]在 Google Maps上畫出簡易的多邊形狀遮罩。

1. 點選右側的「New hole」，在Google Maps上即會產生一個遮罩區域。
2. 拖拉角落的圓點調整區域的範圍。
3. 再次點選「New hole」時則可以產生第2個、第3個區域…。
4. 右手邊複製google maps的code。
